# 변경 이력 (Changelog)

모든 주요 변경 사항은 이 파일에 기록됩니다.

## [Unreleased]

### 추가됨 (Added)
- **홈 화면 (HomeScreen) 식단 기록 기능:**
  - '오후 간식' 카드의 **[기록하기]** 버튼 클릭 시 나타나는 식단 기록 모달(`RecordMealModal`)을 추가했습니다.
  - **금지 식품 섭취 피드백:** 텍스트에 금지 식품(마카롱, 라면, 과자, 술 등)이 포함될 경우, 화면 하단에 빨간색 경고 스낵바(Snackbar)가 노출되며, 해당 식단의 AI 피드백 카드가 즉시 붉은색 경고 테마로 변경됩니다.
  - **뒤늦은 기록 피드백:** 모달에서 '뒤늦게 기록' 체크박스를 선택하면, 노란색(Amber) 스낵바가 노출되고 AI 피드백 카드가 "기록을 잊지 않고 해주셨네요!"라는 격려와 함께 식사 직후 기록을 권장하는 내용으로 업데이트됩니다.
  - **식사 시간 지남 (Missed Meal) 상태 UI:** '저녁' 식사 카드를 통해 식사 시간이 지났으나 아직 입력하지 않은 상태의 UI를 추가했습니다. 노란색(Amber) 경고 테마로 표시되며, [뒤늦게 기록하기] 버튼을 통해 기록을 유도합니다.

### 수정됨 (Fixed)
- **홈 화면 (HomeScreen):** '1일차 모드' 토글 시 `lucide-react`에서 `ChevronRight` 아이콘 임포트가 누락되어 발생하던 렌더링 에러(Crash)를 수정했습니다.
