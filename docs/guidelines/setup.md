# 셋업 화면 (SetupScreen) 가이드라인

## 1. 목적
앱을 처음 실행하거나, 마이페이지에서 데이터를 초기화했을 때 다이어트 시작일을 설정하는 온보딩 화면입니다.

## 2. 주요 기능 및 요구사항

### 2.1. 시작일 선택
*   **커스텀 달력 UI:** 기본 `<input type="date">` 대신, 앱의 디자인 톤앤매너에 맞춘 커스텀 달력(월간 그리드)을 제공하여 직관적으로 날짜를 선택할 수 있게 합니다.
*   **시각적 안내:** 스위치온 다이어트 4주 프로그램을 시작한다는 동기부여 문구와 아이콘을 배치합니다. ("박용우" 등 특정 인물 이름은 제외하고 범용적인 문구 사용)
*   **팁 제공:** "오늘부터 바로 시작하는 것을 추천해요! 1~3일차는 단백질 쉐이크만 섭취해야 하므로 주말을 끼고 시작하는 것도 좋은 전략입니다." 와 같은 실용적인 팁을 제공합니다.

### 2.2. 다이어트 시작하기 버튼
*   **동작:** 사용자가 날짜를 선택하고 버튼을 누르면, 해당 날짜를 기준으로 28일 챌린지가 세팅되고 `HomeScreen`으로 이동합니다.
*   **디자인:** 크고 명확한 초록색 버튼으로 디자인하여 사용자의 행동을 유도합니다.
