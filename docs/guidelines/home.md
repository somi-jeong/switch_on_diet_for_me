# 홈 화면 (HomeScreen) 가이드라인

## 1. 목적
사용자가 매일 앱을 켰을 때 가장 먼저 마주하는 화면으로, '오늘 하루'의 다이어트 미션을 직관적으로 파악하고 수행할 수 있도록 돕습니다.

## 2. 주요 기능 및 요구사항

### 2.1. 상단 요약 카드
*   **현재 상태 표시:** "1주차 4일째" 등 현재 다이어트 진행 단계를 명확히 보여줍니다.
*   **공복 타이머:** 저녁 식사 후 14시간 공복을 유지하는 타이머를 시각적으로 표시합니다. (프로그레스 바 활용)

### 2.2. 데일리 미션 체크리스트
스위치온 다이어트의 핵심 규칙을 매일 체크할 수 있도록 구성합니다.
*   **물 2L 이상 마시기:** 버튼 클릭으로 토글
*   **영양제 챙겨먹기:** 버튼 클릭으로 토글
*   **고강도 운동하기:** 주 4회 이상, 숨이 찰 정도의 운동 여부 체크
*   **수면 시간 (자동 기록):** 기상 시간과 취침 시간을 기반으로 자동 계산되어 표시됩니다. 6시간 이상이면 성공, 미만이면 경고 아이콘이 나타납니다.

### 2.3. 식단 타임라인
*   **4단계 식사 스케줄:** 아침, 점심, 간식, 저녁의 권장 식사 시간을 표시합니다.
*   **기록 상태 시각화:** 이미 기록된 식단은 초록색 체크, 아직 기록 전인 식단은 회색 플러스 버튼으로 구분합니다.
*   **기상 전 상태:** 사용자가 '지금 일어났어요' 버튼을 누르기 전까지는 식단 타임라인이 비활성화(흑백 처리)되어야 합니다.

### 2.4. 수면 및 기상 관리
*   **취침 시간 경고:** 자정~새벽 4시 수면 규칙을 지키기 위해, 밤 11시 30분경 취침 알림 모달을 띄웁니다.
*   **기상 시간 지연 경고:** 기상 시간이 너무 늦어(오전 9시 이후) 아침 식단을 건너뛰어야 할 경우, 공복 유지 및 점심 식사 안내 모달을 제공합니다.
*   **극단적 기상 지연 (오후 3시 이후):** 기상 시간이 극단적으로 늦어질 경우, 정규 식단 스케줄(아침, 점심, 간식)을 모두 취소하고 가벼운 단백질 쉐이크 1잔만 섭취하도록 안내하는 전용 UI로 전환됩니다.
*   **1일차 수면 시간 입력:** 다이어트 첫날에는 전날 취침 기록이 없으므로, 홈 화면의 수면 카드 클릭 시 '전날 취침 시간'을 직접 입력받아 수면 시간을 계산합니다.

### 2.5. 오늘의 AI 코치 피드백
*   **목적:** 캘린더 화면에 들어가지 않고도, 오늘 하루의 규칙 달성도에 대한 AI 코치의 피드백을 홈 화면에서 바로 확인할 수 있습니다.
*   **구성:** 피드백 등급(완벽 성공, 보통, 최악), 코멘트, 그리고 현재 놓치고 있는 규칙(예: 기상 시간 지연 등)을 표시합니다.

### 2.6. 식단 기록 및 실시간 피드백 (Snackbar & AI 코치)
*   **금지 식품 섭취 시:** 사용자가 식단 기록 시 금기 식품(예: 마카롱, 라면, 과자, 술 등)을 입력하면, 화면 하단에 **빨간색 경고 스낵바(Snackbar)**가 즉시 나타납니다. 동시에 해당 식단의 AI 코치 피드백 카드가 빨간색 테마로 변경되며, 인슐린 분비와 지방 분해 중단에 대한 강력한 경고 메시지를 제공합니다.
*   **뒤늦은 기록 시:** 식사 시간보다 한참 뒤에 늦게 기록한 경우(모달 내 '뒤늦게 기록' 체크), 화면 하단에 **노란색(Amber) 스낵바**가 나타납니다. AI 코치 피드백은 "기록을 잊지 않고 해주셨네요! 다음부터는 식사 직후에 기록하면 더 정확한 코칭을 받을 수 있어요."라며 격려와 올바른 습관을 안내합니다.
*   **정상 기록 시:** 허용 식품을 제때 기록하면 **검은색/초록색 성공 스낵바**가 나타나며, AI 코치가 긍정적인 피드백을 제공합니다.

## 3. 트러블슈팅 및 알려진 이슈 (Troubleshooting)
*   **아이콘 임포트 누락 에러:** UI 컴포넌트(예: 1일차 수면 시간 입력 모달 진입 버튼)에 `lucide-react` 아이콘(예: `ChevronRight`)을 추가할 때, 상단 import 문에 해당 아이콘이 포함되어 있는지 반드시 확인해야 합니다. 누락 시 렌더링 에러(Crash)가 발생합니다. (해결됨)
